# ربات عضویت خودکار در گروه‌های روبیکا

این ربات با استفاده از کتابخانه `pyrubi` توسعه داده شده و قابلیت‌های زیر را ارائه می‌دهد:

- عضویت خودکار در گروه‌ها از طریق لینک‌های موجود در کانال‌های لینکدونی
- ارسال پیام‌های دوره‌ای به گروه‌های عضو شده با قابلیت تنظیم زمان‌بندی
- پاسخ‌دهی خودکار به پیام‌های خصوصی با دو سطح پاسخ (پیام خوش‌آمدگویی و پاسخ ثانویه)
- پشتیبانی از چندین سشن همزمان
- ذخیره‌سازی پیشرفت و امکان ادامه از نقطه توقف
- مدیریت گروه‌ها با قابلیت خروج دستی
- تنظیمات پویا از طریق دستورات متنی

## پیش‌نیازها

- Python 3.8 یا بالاتر
- اکانت فعال روبیکا با سشن معتبر (فایل‌های `.session`)
